# Multilingual Teacher Evaluation Project

In this project, we ask the question: "what makes a good multilingual teacher for off-policy distillation?" Specifically, we propose a metric that evaluates the quality of data generated by the teacher model *(intrinsic)* and the performance gain of the student model on some benchmarks *(extrinsic)*.

## Setup & Installation

Make sure that you have `uv` in your system (see [download instructions](https://docs.astral.sh/uv/getting-started/installation/)).
To install all dependencies, run the following command:

```sh
uv sync --dev
```
